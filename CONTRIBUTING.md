# Katkıda Bulunma Rehberi

Bu proje kişisel bir seyahat günlüğüdür, ancak önerilere ve düzeltmelere açıktır.

1. Bu repoyu fork'layın.
2. Yeni bir dal (branch) oluşturun (`git checkout -b ozellik/YeniOzellik`).
3. Değişikliklerinizi yapın ([_Sablon](_Sablon) klasörünü kullanmayı unutmayın).
4. Değişikliklerinizi commit'leyin (`git commit -m 'Yeni şehir eklendi: X'`).
5. Dalınızı push'layın (`git push origin ozellik/YeniOzellik`).
6. Bir Pull Request oluşturun.
